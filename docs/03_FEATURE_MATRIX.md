# WorkNow - Feature Matrix

This matrix maps the core features required for the MVP launch of WorkNow across all three user roles.

## Worker Features
*   `[x]` **Register:** Sign up with phone/email and select user role.
*   `[x]` **Login:** Authentication interface.
*   `[x]` **Verification:** Mock National Identification Number (NIN) submission and status badge.
*   `[x]` **Browse Jobs:** Search and filter open casual jobs nearby.
*   `[x]` **Apply:** Quick-apply mechanism for open jobs.
*   `[x]` **Active Jobs:** Manage current active engagements and job details.
*   `[x]` **Wallet:** View current earnings balance, payout history, and mock OPay transfer status.
*   `[x]` **Ratings:** Double-blind feedback system and scorecards.
*   `[x]` **SOS:** In-app safety emergency trigger for active gigs.

---

## Employer Features
*   `[x]` **Register:** Sign up with phone/email and select user role.
*   `[x]` **Login:** Authentication interface.
*   `[x]` **Verification:** Identity profile checking template.
*   `[x]` **Create Job:** Job creation portal with location details and budget/pay fields.
*   `[x]` **Fund Escrow:** Lock cash for a job into the mock escrow wallet using OPay.
*   `[x]` **Manage Applicants:** Review, accept, or decline applications.
*   `[x]` **Release Payment:** Trigger payout of escrow funds to worker on job completion.

---

## Admin Features
*   `[x]` **User Management:** Access profiles, toggle ban status, and edit details.
*   `[x]` **Job Management:** Audit listings, close inappropriate posts.
*   `[x]` **Disputes:** Mediation portal to resolve client-worker payout conflicts.
*   `[x]` **Safety Alerts:** Monitor active SOS signals, check worker coordinates.
*   `[x]` **Verification Queue:** Manually review and verify mock NIN submissions.
*   `[x]` **AI Monitoring:** Log flagged messages scanned by the mock Gemini engine.
