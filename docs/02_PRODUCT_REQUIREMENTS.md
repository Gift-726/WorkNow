# WorkNow - Product Requirements

## 1. User Roles
1.  **Worker:** Users seeking casual work, applying for gigs, completing tasks, and receiving payouts.
2.  **Employer:** Users posting casual tasks, funding escrow milestones, hiring workers, and releasing payments.
3.  **Admin:** System moderators reviewing user registrations, transactions, disputes, safety flags, and moderation logs.

---

## 2. Worker Features
*   **Register:** Sign up using phone number and email.
*   **Login:** Secure mobile login using password or mock OTP.
*   **Select Worker Role:** Set active profile view as a worker.
*   **Complete Profile:** Add skills, profile picture, location, and bio.
*   **View Jobs:** Browse the local gig board.
*   **Search Jobs:** Find gigs via keywords or category tags.
*   **Filter Jobs:** Filter jobs by daily pay rate, distance, and category.
*   **Apply for Jobs:** Submit simple applications for posted gigs.
*   **Track Applications:** View state of submitted bids (Pending, Accepted, Rejected).
*   **View Active Jobs:** Monitor ongoing contracts that are funded and ready to start.
*   **Complete Jobs:** Mark active contracts as done, uploading mock delivery evidence.
*   **View Payment History:** Check earnings and status of funds (Held in Escrow, Released to Wallet).
*   **View Ratings:** Access reviews and star-ratings received from employers.
*   **Use SOS Button:** Access the critical emergency help option during active jobs.

---

## 3. Employer Features
*   **Register:** Sign up using phone number and email.
*   **Login:** Secure login interface.
*   **Select Employer Role:** Toggle profile view to create listings.
*   **Complete Profile:** Setup business or household identity.
*   **Create Jobs:** Post casual gigs specifying title, daily pay, location, and description.
*   **Edit Jobs:** Update or delete draft job listings.
*   **Fund Escrow:** Lock down the gig fee via mock OPay payment portal before worker starts.
*   **Review Applicants:** Evaluate applicant cards, ratings, and profile histories.
*   **Accept Workers:** Confirm hiring of chosen applicants.
*   **Track Jobs:** Monitor current worker status (Not Started, In Progress, Completed).
*   **Approve Completion:** Confirm work is done and matches requirements.
*   **Release Payments:** Transfer held funds from mock escrow wallet to the worker.

---

## 4. Admin Features
*   **View Users:** Oversee worker and employer accounts, check verification status.
*   **View Jobs:** Monitor active and historical casual job posts.
*   **View Disputes:** Review payments frozen due to disagreements and decide payouts.
*   **View Safety Alerts:** Monitor triggered SOS emergency flags and location warnings.
*   **View Verification Requests:** Approve or reject mock NIN verification documents in the queue.
*   **View AI Moderation Logs:** Monitor flagged chats, descriptions, or reviews processed by mock Gemini.
